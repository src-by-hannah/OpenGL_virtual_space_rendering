# OpenGL_virtual_space_rendering

## Install
* freeglut library

<br>

## Function
1. Change the rendering mode

  - Wireframe

  - Flat Shading

  - Smooth Shading

  - Back-Face Culling

<br>

2. Change the texturing option

  - Point Sampling

  - Linear Sampling

  - Clamp

  - Repeat

  - Mapping (Decal / Replace / Modulate / Blend)

<br>

3. Change the Projection mode

  - Parallel

  - Perspective

<br>
4 Light On/Off
