## Function

<b>1. Change the rendering mode</b>

  - Wireframe

  - Flat Shading

  - Smooth Shading

  - Back-Face Culling

<br>

<b>2. Change the texturing option</b>

  - Point Sampling

  - Linear Sampling

  - Clamp

  - Repeat

  - Mapping (Decal / Replace / Modulate / Blend)

<br>

<b>3. Change the Projection mode</b>

  - Parallel

  - Perspective

<br>

<b>4. Light On/Off</b>

<br>

<b>5. Change the Material</b>

  - Gold

  - Bronze
  
  - Chrome
  
  - Cooper

<br>

<b>6. Navigation(Using the keyboard)</b>
